Project Overview
Customer churn is a major challenge for businesses such as banks, telecom companies, and subscription services. Losing customers results in revenue loss and increased costs to acquire new ones.
This project focuses on generating synthetic customer data and performing Exploratory Data Analysis (EDA) to identify patterns, trends, and key factors influencing customer churn.
________________________________________
 Objectives
•	Generate a large synthetic dataset (100,000 customers) 
•	Perform data cleaning and preprocessing 
•	Analyze customer behavior using statistical methods 
•	Identify high churn and low churn segments 
•	Perform group-based and relationship analysis 
•	Visualize data using graphs and charts 
•	Extract insights to reduce churn
